Tentang Saya

Halo! Saya Muhammad Ilham Jagad. Perjalanan saya di dunia programming dimulai dari rasa penasaran sederhana: "Bagaimana bisa komputer itu memahami bahasa manusia?"

Seiring waktu, rasa penasaran itu berkembang menjadi hasrat mendalam untuk menciptakan solusi teknologi yang menggabungkan kecerdasan buatan dengan aplikasi praktis. Saya telah mengasah keterampilan saya dalam AI Engineering, Backend System, dan pengembangan Mobile App.

Saya percaya bahwa teknologi memiliki potensi luar biasa untuk mengubah cara kita hidup dan bekerja, dan saya berkomitmen untuk menjadi bagian dari perubahan itu.
